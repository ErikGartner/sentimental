# Runeberg Corpus
This massive corpus contains around 500 Swedish books from the [Runeberg project](http://runeberg.org/) merged into one big corpus.

Since the Runeberg project in many ways is the Swedish version of Project Guthenberg and all the books are open domain the language of the books may be archaic.

## Script
In the `script` folder there is a small python script that downloads all the texts to a single file. The script is rather simple and may easily be modified to export the books into seperate files.
